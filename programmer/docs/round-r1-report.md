# Programmer Skill — R1 优化报告

> 日期: 2026-04-30
> 轮次: R1（指令具体性）
> 基线: 67.4 → 改进后: ~74（含 grep 同类修复）

## 改动摘要

| 改动 | 旧 | 新 | 解决的问题 |
|------|----|----|-----------|
| 复杂度三档 | "中等改动"无量化 | 1文件/≤50行, 2-5文件/50-300行, ≥6文件/>300行 + 产出列 | 维度5: 模糊判定 |
| Bug Fix quick | 3步（根因→修复→验证） | 4步（grep同类→根因→修复→验证） | 维度8: 同类遗漏 |
| Step 1 加载 | 伪代码块 | 有序号可执行步骤 | 维度2: 执行歧义 |
| 约束注入 | 伪代码+示例 | 可填充模板 `{从memory加载}` | 维度5: 不可执行 |
| 规范存储 | 无示例 | 3个真实示例+去重策略 | 维度5: 粒度模糊 |
| 首次使用 | "提示用户" | 标准化输出格式模板 | 维度8: 输出不确定 |

## 评估者反馈

> "维度5从6跳到8，复杂度量化+模板+示例解决了核心模糊性。维度8提升一档，首次使用格式和产出列对高频场景有直接帮助。"
