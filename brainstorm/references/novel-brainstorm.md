## 小说专用创意工具

### 世界观引擎

**冰山模型**：读者看到10%，作者构建100%。先写完整世界规则，再决定每卷暴露多少。通过日常场景侧面传达，禁止exposition倾倒。

**核心意象衍生**：从一个核心意象像涟漪扩展。核心意象 → 第二层(社会/经济/政治) → 第三层(文化/禁忌/日常)。每层问"这个设定会导致什么现实后果？"

**极端化推演**：从核心设定推演第二层、第三层社会影响。例：记忆交易 → 富人买体验 → 记忆版权法 → 国家用虚假记忆洗脑。

**代价守恒律**：任何超能力必须有等价且被社会接受的代价。代价太小 → 增大直到让使用者犹豫。抽象代价 → 转化为身体/社会/心理后果。

**感官地图**：为关键场景建五感档案，与情绪基调**相反或错位**。屠杀后烤面包香 / 恐怖地下室干净消毒水味。

### 势力冲突引擎

**博弈局面法**（最少三方）：两方是对抗，三方以上才有博弈。每个势力定义：核心资源 / 核心恐惧 / 核心目标 / 独立时间线。找资源交叉点：A的废品是B的神级宝物。

**阵营四要素**：信仰原则 / 行事方法 / 习俗(节日/礼仪/禁忌) / 语言(术语/表达习惯)。Monolith警告：真实势力内部一定有派系分歧。

**信息不对称引擎**：每个关键冲突点，列出各方"知道什么"和"不知道什么"，用信息差制造戏剧性。

**冲突个人化**：冲突不能只是概念，必须有血和伤疤。矿奴咳灵晶碎片（不是"被剥削"，是肺在碎裂）。

### 力量体系引擎

**Sanderson三定律**：
1. 局限 > 能力 — 限制比能力更有趣
2. 局限 > 创新 — 在现有能力限制上做文章，而非加新规则
3. 先扩展后增加 — 先用好已有元素，再添加新元素

**力量体系 = 冲突引擎**：删掉升级机制，故事还有张力吗？不同等级的社会地位差异是日常冲突来源。力量的获取途径和代价由谁控制？

**性格决定能力**（HxH模式）：能力是性格的延伸。自我施加的限制 = 叙事杠杆。角色选择制约条件本身就揭示其价值观。

### 角色引擎

**欲望三角**：Want(表面欲望) / Need(深层需求) / Fear(得到Need后失去什么身份认同)。所有关键选择都在这三者间撕裂。

**潜台词冰山**：对话三层——字面意思(10%) / 社交意图(30%) / 真实情感(60%)。只写水面上的10%。

**POV透镜过滤**：同一场景换三个视角，每个版本的偏见/秘密/感官缺陷扭曲事实。

**主题物化法**：将抽象主题转化为具体物品/身体疾病/重复动作。主题"遗忘" → 只能记住24小时的病症 / 不断修补裂缝的镜子。

### 剧情引擎

**多米诺因果链**：不先想"发生什么"，先想"因为前一件事，所以不可避免发生后一件事"。每步因果之间没有"巧合"缝隙。

**虚假胜利/失败**：转折点设计看似达成目标实则埋下更大危机的事件。以为杀死反派实则帮其完成仪式。

**限制爆破法**：给主角施加极端且不可违背的限制，在限制内找解。限制越荒谬创意越锋利。

**延迟引爆**：第一章放置看似无关的道具/细节/角色，终章以变形方式回收。墙上旧猎枪 → 结尾用枪托砸碎玻璃逃生。

### 反派引擎

**六维度模型**：情感锚定(失去什么) / 行为反差(对不同人不同面) / 价值观平行(和主角追求同一目标路径不同) / 脆弱性场景(什么时候动摇) / 情感梯度(与主角关系的曲线变化) / 结局留白(恨不起来)。

**合理化辩护**：用反派逻辑写一段能说服自己的演讲，不用"疯狂""邪恶""天生坏种"。说服了自己 → 反派设计成功。

### 思维工具箱（解卡专用）

- **"如果相反"反转术**：将最理所当然的设定直接反转
- **陌生化组合**：两个熟悉但从未并置的元素强行组合，合理化共存
- **随机刺激法**：从无关领域随机抽取概念强制嫁接

## 小说领域六项核心原则

头脑风暴过程中始终检查：

1. **差异化 = 张力 = 戏剧性** — 它和别人有什么本质不同？
2. **信息管理 = 叙事引擎** — 什么时候让读者知道什么？
3. **自洽 = 最高原则** — 自洽产生必然感，必然感 = 读者信服的基础
4. **限制 > 能力** — 限制比能力更有趣，稀缺性比丰富性更有张力
5. **世界观-势力-反派三位一体** — 三者不能分开设计，互相验证
6. **NPC有自己的生活** — 配角不为剧情服务，有自己的逻辑和时间线
