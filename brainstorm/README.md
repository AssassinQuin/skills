# Brainstorm

Structured brainstorming with three phases: DIGGING (deep-dive into real problem) → INCUBATING (cross-domain collision) → CONVERGING (filter & rank). Includes novel-writing domain enhancement.

## Trigger

头脑风暴、brainstorm、brainstorming、碰撞、发散、创意、想法、帮我想想

## Quick Start

```
/brainstorm 头脑风暴如何提升 API 性能
/brainstorm 帮我想一个 SaaS 产品方向
/brainstorm 怎么让用户留存率翻倍
```

## Workflow

1. **Input Validation** — Check if brainstorming is the right tool for this request.
2. **DIGGING** — Deep-dive into the real problem behind the surface request.
3. **INCUBATING** — Cross-domain collision with stimulus pool (random stimulus, analogies, inversions).
4. **CONVERGING** — Filter, rank, and select best ideas.
5. **User Correction** — Built-in mechanism for user to redirect at any phase.

### Novel Writing Enhancement

For fiction/creative writing requests, activates `novel-brainstorm.md` with domain-specific techniques (character arcs, plot structures, world-building).
