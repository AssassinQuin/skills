# 01-网文大纲方法论与模板流程

## 一、大纲写作方法论

### 1.1 三幕式结构（最通用）

源自好莱坞百年实践，比例25%-50%-25%：
- **第一幕（25%）：建置** — 角色介绍+世界观引入+触发事件
- **第二幕（50%）：对抗** — 递增阻碍+中间点转折+最低谷
- **第三幕（25%）：解决** — 成长后解决危机

每幕可映射为一个或多个卷。马良写作将其适配为网文框架。[来源: maliangwriter.com]

### 1.2 升级打怪循环（网文独有）

非传统故事结构，而是循环叙事模式：
- **弧段流程**：新地图→低调→结仇→修炼→打脸→奖励→更大舞台
- **经典节奏**：受辱→立誓→修炼→变故→谷底→突破→打脸（7步）
- **关键约束**：每个循环必须有差异化元素，否则纯套路导致读者疲劳
- **节奏红线**："受辱→立誓→修炼"不能超过3章，否则读者流失

《斗破苍穹》（天蚕土豆，2009）将修行等级细分为十级（斗者→斗帝），每阶九星。[来源: maliangwriter.com + zhihu.com]

### 1.3 雪花法（Snowflake Method）

物理学家/作家Randy Ingermanson发明，灵感来自数学分形原理。核心优势：任何阶段都可回头修改。

**10步流程**：
1. 一句话概括（核心概念）
2. 一段式概括（5句话涵盖起承转合）
3. 一页纸人物介绍
4. 一页纸大纲
5. 人物大纲（补充动机/冲突/成长弧）
6. 四页纸大纲
7. 完整人物表
8. 场景清单
9. 场景描述
10. 开始写初稿

Notion已有包含所有模块的现成模板。[来源: advancedfictionwriting.com + reedsy.com + notion.com]

### 1.4 Save the Cat 15节拍表

Blake Snyder（好莱坞编剧，2005年出版）将三幕剧细化为15个精确节拍：
- 开场画面(1%)→主题陈述(5%)→铺垫(1-10%)→催化剂(10%)→辩论(10-20%)
- 进入第二幕(25%)→B故事(22%)→游戏时间(20-50%)→中点(50%)
- 坏人逼近(50-75%)→一无所有(75%)→灵魂黑夜(75-85%)
- 进入第三幕(85%)→终局(85-110%)→终场画面(100%)

Snyder名言："如果不能一两句话填完一个节拍——你还没有节拍，只是在踩水。"

与三幕式互补：三幕式搭骨架，节拍表填血肉。[来源: reedsy.com + savethecat.com]

### 1.5 行业共识的大纲粒度

| 层级 | 字数 | 内容 |
|------|------|------|
| 总纲 | 1000-2000字 | 整书主线走向 |
| 卷纲 | 200-500字/卷 | 当前弧段详细规划 |
| 章节蓝图 | 50-200字/章 | 具体内容指令 |

推荐工作流：选模板写总纲（1-2小时）→拆成5-10卷写卷纲→写当前卷的章节蓝图→正文生成+审核。[来源: maliangwriter.com + write.qq.com]

## 二、大纲模板与实操流程

### 2.1 标准大纲模板结构

基于橙瓜/阅文/简书多源综合：

```
一、书名与备选书名
二、作者笔名
三、类型与字数
四、简介（吸引读者）
五、核心设定
   - 金手指/能力体系
   - 世界格局（广义格局/传说格局/势力划分）
   - 力量体系（升级体系/原理/差距）
六、主要人物（姓名/性别/年龄/外貌/社会地位/认知水平/关系）
七、全书概要
   - 序幕→开端→发展→高潮→尾声
   - 支线：启动条件/持续时间/主要作用/结果
八、需要用到的描写手法
```

### 2.2 升级打怪参数化模板

每卷定义4个核心变量：

| 参数 | 卷一 | 卷二 | 卷三 |
|------|------|------|------|
| 当前地图/势力 | 出生地图 | 宗门 | 秘境 |
| 主角等级 | 练气期 | 筑基期 | 金丹期 |
| 反派级别 | 地方恶霸 | 内门长老 | 其他宗门天才 |
| 差异化元素 | — | 失散兄弟 | 双重身份 |

每卷等级和舞台翻倍。

### 2.3c 人物大纲七要素法

知乎专栏"枫轻谈创作技巧"强调：**小说不是由情节塑造人物，而是以人物为中心去设计情节**。人物大纲应包含：

1. **人物概述**：最初脑海中对人物的朦胧轮廓
2. **人物设定**：身世背景等
3. **人物重大人生经历**：不一定要出现在作品中，但对人物塑造不可或缺（可写番外）
4. **关联人物关系图**：以主角为核心的辐射关系图
5. **细致的性格分析**：从人物性格矛盾入手，划分幼年/少年/青年/中年阶段

**实操建议**：用两个A5笔记本——正面写主角，背面写龙套；另一本正面写剧情/主线/地图/世界局势，背面写分卷人物关系和章节大纲。写完三卷大纲就可以开始创作。[来源: zhihu.com/枫轻谈创作技巧]

### 2.3 大纲"填空题"模板

B站作者"网文徐掌柜"提出的入门模板：
> [主角]在[地点]因为[事件]被迫[行动]，途中遇到[阻碍]，通过[方法]克服，最终[结果]。

### 2.4 故事链条法（从细纲到小剧情）

知乎作者"修北"提出故事链条概念：将小剧情拆解为环环相扣的环节，每个环节都可能产生剧情点。

**方法**：
1. 确定小剧情目标（如"主角通过家教赚钱"）
2. 列出完成目标所需的所有环节（主角想法→传播信息→找到学生→准备资料→获得认可→开始担任→成绩提升→获得资金）
3. 筛选能产生冲突/剧情的环节（如何获得认可？如何找到学生？）
4. 不出剧情的环节一笔带过

**3/5/7 节奏法**：将筛选后的环节组织为三种复杂度：
- **3拍**：引出事件→解决事件→引出下一个事件（过场剧情/日常补充）
- **5拍**：引出→参与→强冲突→解决→引出下一个事件（标准网文写法，最推荐）
- **7拍**：引出→参与→强冲突→以为解决→再冲突→真正解决→引出（复杂剧情，慎用）

这套节奏法可嵌套使用——3/5/7用于小剧情，也适用于主线和大纲设计。[来源: zhihu.com/修北]

### 2.5 鱼骨图主线法（信息量原则）

**核心原则**：大纲的信息量决定故事的丰富度。写主线时，一句话应包含全书所有信息量。

**鱼骨结构**：
- 主干（主线）：讲述故事本身，塑造主角形象
- 鱼刺（支线）：辅助补充世界观，塑造配角形象
- 设计顺序：先考虑主线，再融入支线

**阶段性目标分解**（以150万字重生创业为例）：
```
主线6阶段：第一桶金→财富爆发→市首富→省首富→全国首富→世界首富
支线1：弥补前世家庭遗憾
支线2：改善自己的生活
```

**关键**：所有文字只做两件事——推进主线，或树立人设。掌握这两点就不会水文。[来源: zhihu.com/修北]

### 2.6 不同类型适配要点

| 类型 | 大纲核心需求 |
|------|------------|
| 升级流 | 等级-地图-反派参数表 |
| 悬疑推理 | 线索-红鲱鱼-揭示时机倒推表 |
| 群像叙事 | 多线时间轴+交汇节点网格表 |
| 无限流 | 主线进度+副本参数+成长双维度汇总表 |

### 2.5 AI辅助大纲工作流

行业正在形成的共识：
- 步骤1-2（一句话/段落）：AI辅助概括和扩展
- 步骤3-5（角色）：AI生成角色档案
- 步骤8-9（场景清单/描述）：AI批量生成
- **核心原则**：人类做创意决策，AI做执行扩展

### 💡 灵感

1. **"三层嵌套"模型**：全书三幕式骨架 + 卷内STC节拍表 + 弧段内升级循环。三层独立可调
2. **参数化卷模板**：为每卷定义固定参数字段，AI直接读取参数表生成，将"创意写作"降维为"参数配置"
3. **逆向雪花法**：写完的卷逆向压缩（场景→一页→一段→一句），用于快速生成卷摘要和检测偏离
4. **3/5/7节奏嵌入AI**：AI生成章节蓝图时，可自动标注每章的节奏类型（3/5/7），保证剧情密度均匀分布
5. **故事链条自动拆解**：给定一个小剧情目标，AI可自动拆解为故事链条环节，并标注冲突潜力点
