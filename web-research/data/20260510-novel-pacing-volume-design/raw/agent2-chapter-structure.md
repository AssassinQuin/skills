# Agent2: 卷内章节结构 - 原始输出

## 一、一卷通常多少章

| 来源 | 每卷字数 | 章节字数 | 每卷章节数 |
|------|----------|----------|------------|
| 知乎(53413850) | 10万字 | 2000字 | **50章** |
| 飞书指南 | 5-6万字 | 2500-3000字 | **20-30章** |
| 马良写作 | 3-8万字 | 3000-5000字 | **10-25章** |
| 中国作家网 | - | 5000-8000字 | - |

**结论**: 以3000字/章计，大约10-30章/卷。高潮卷可能偏短密集爆发，过渡卷偏长拉日常铺垫。

## 二、爽点三级密度（马良写作）

| 爽点级别 | 频率 | 内容 |
|----------|------|------|
| 小爽点（每章） | 1章1个 | 小胜利/小反转/解锁奖励 |
| 中爽点（3-5章） | 每3-5章1个 | 完整打脸/关系突破/实力跃升 |
| 大爽点（每卷） | 每卷高潮1个 | Boss战逆转/身份曝光/多伏笔回收 |

- 追读率top10%作品平均每1.8章有一次情绪高峰
- bottom10%平均间隔4.7章
- **平路不超过3章**: 超过3章无爽点，追读就开始掉

## 三、单章结构模板（起承转爽）
- 起500字(25%) + 承800字(40%) + 转300字(15%) + 爽400字(20%)
- 章末钩子4种类型: 悬念式、反转式、情绪炸弹式、信息投放式

## 四、三级大纲体系（马良写作）
- 总纲 → 卷纲 → 章节蓝图
- 每卷3-8万字
- 卷纲: 核心事件2-3件 + 主角状态变化 + 收尾钩子
- 章节蓝图500-800字: 开场→发展(3-5节点)→转折→结尾+钩子

## 五、黄金5章模板（飞卢）
- 第1章钩子→第2章升温→第3章小爆发→第4章反转→第5章大爆发+新钩子
