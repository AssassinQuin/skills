# Storytelling (故事創作)

故事創作技能：小說寫作、漫畫腳本、影片企劃、敘事結構、角色塑造。繁體中文為主，支援多種故事結構模板。

## Trigger

故事、小說、劇本、漫畫腳本、影片企劃、storytelling、故事創作、角色設計

## Quick Start

```
/storytelling 寫一個科幻短篇小說，關於記憶可以交易的世界
/storytelling 幫我設計一個漫畫角色的成長弧線
/storytelling 規劃一部 YouTube 系列影片的敘事結構
```

## Workflow

1. **Step 1: 需求診斷** — 確認故事類型、篇幅、目標讀者
2. **Step 2: 結構設計** — 選擇敘事結構（三幕劇、英雄旅程、起承轉合等）
3. **Step 3: 角色構建** — 塑造角色動機、衝突、成長弧線
4. **Step 4: 逐段創作** — 按結構分段撰寫，保持節奏
5. **Step 5: 自檢** — 對照結構模板檢查一致性

## 支援的故事結構

| 結構 | 適用場景 |
|------|---------|
| 三幕劇結構 | 電影、小說、短篇 |
| 英雄旅程 (Hero's Journey) | 冒險、成長故事 |
| 起承轉合 | 中文敘事傳統 |
| 非線性敘事 | 懸疑、文學作品 |
