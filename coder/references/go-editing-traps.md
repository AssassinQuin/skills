# Go 编辑陷阱

## Tab 缩进问题

Go 文件用 tab 缩进。Edit 工具可能无法匹配 tab 内容（Read 输出中 tab 渲染为空格）。

## 操作推荐表

| 操作 | 推荐方式 |
|------|----------|
| 删除行 | `sed -i '' 'Nd' file` |
| 替换块 | 先 `cat -v` 确认空白 → `sed` 替换 |
| 写新文件/重写文件 | `Write` 工具（不受 tab 匹配影响） |
| 精确插入 | `Read` 获取行号 → `sed -i '' '{行号}a\内容' file` |

## 文件归属决策

| 文件类型 | 是否可编辑 | 原因 |
|----------|-----------|------|
| `dao/` | **不可** | `make dao` 自动生成 |
| `service/` 接口方法签名 | **不可** | `make service` 自动生成 |
| `model/entity/` | **不可** | `make dao` 自动生成 |
| `model/do/` | **不可** | `make dao` 自动生成 |
| `logic/` | **可** | 手写业务逻辑 |
| `controller/` | **可** | 手写 HTTP 处理 |
| `consts/` | **可** | 手写常量定义 |
| `model/dto/` | **可** | 手写层间传输对象 |

## 反模式

| 反模式 | 正确做法 |
|--------|----------|
| Edit 编辑 tab 缩进文件 | `sed` 或 `Write` |
